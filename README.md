## Building and Running ##

**Run:**
 

    npm install
    electron .

**Build:**

    npm install
	npm install --save-dev electron
    npm install electron-packager --save-dev
 Windows:
 

	npm run build-win
  macOS:
  

	npm run build-mac
   Linux:
   

	npm run build-linux
